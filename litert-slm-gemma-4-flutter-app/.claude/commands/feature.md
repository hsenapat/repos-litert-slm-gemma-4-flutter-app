# /feature

Scaffold a new feature in this Flutter app.

## Steps

1. Create model in `lib/models/<name>.dart` if new data type is needed.
2. Add service logic in `lib/services/` or extend `gemma_service.dart`.
3. Add/update screen in `lib/screens/<name>_screen.dart`.
4. Add widgets in `lib/widgets/` if reusable UI is needed.
5. Wire up routing in `main.dart` if a new screen.
6. Update `docs/Features.md`.
